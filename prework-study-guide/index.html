HTML

  • HTML means Hypertext Markup Language.
  • The head element contains information, or Metadata about the webpage.
  • The body element represents the visible content shown to the user.
  • The img element contains information about images that are displayed on the webpage.
  • An "element" refers to the entire element from opening tag to closing tag.
  • A "tag" refers only to what appears inside the angle brackets.

CSS

  • CSS stands for "Casccading Style Sheet"
  • CSS Defines the webpage's style such as: colors, aesthetics, visual laylout, and organization.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

  • Git is a version control system that records the changes made to our code.
  • Allows us to go back to previous versions of our code to fix any mistakes that may arise.
  • git status: Checks what branch we are currently on.
  • git checkout -b branch-name: Creates a new branch and switches to it.

JavaScript

  • JavaScript (JS) is a high-level and dynamic programming language that enables interactivity with a website.
  • A variable is a named container that allows us to store data in our code.
  • To declare a variable, we use the var keyword. After the var keyword, we give the variable a name.
  • Control flow is the order in which a computer executes code in a script.
  • JavaScript is at work whenever users click a button, retrieve data, or filling out a form on a webpage.
